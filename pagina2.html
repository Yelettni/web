Ordenación por Burbuja

El método de ordenamiento conocido como método de la burbuja consiste en tomar el último elemento, compararlo con el anterior y, de ser necesario, realizar el intercambio para que el elemento menor siempre quede adelante. Luego, se toma el penúltimo elemento, se compara con el anterior y, de ser necesario, se realiza el intercambio, y así sucesivamente, hasta que el menor elemento de todo el array se encuentre en la primera posición. Como el elemento menor ya está en su sitio, este debe obviarse en los siguientes pasos; por lo tanto, se repite todo el proceso, pero sin considerar los elementos que van quedando en su sitio.
Al inicio debemos leer los n elementos en el array x[i]. En los bucles anidados, el bucle más interno (el bucle j) recorre todos los elementos del array desde la última posición (n-1) hasta la última ubicación ordenada (i). En la primera iteración irá comparando cada elemento con el anterior desde n-1 hasta i=1. Al llegar al elemento con subíndice j=1 (j>=i) se hace la comparación x[0]>x[1], con lo cual aseguramos que fueron comparados todos los elementos. El elemento menor de todos ellos se encuentra en x[0]. Luego i se incrementa en 1, para que en el bucle j el elemento x[0], al estar ordenado, no sea tomado en cuenta. Posteriormente se recorren los elementos sin ordenar, desde la última posición (n-1) hasta la última posición no ordenada i, comparándolos e intercambiándolos hasta finalizar el bucle i; esto es, hasta que todos los elementos estén ordenados o hasta que la última posición no ordenada sea la (n-1). Finalmente, imprimimos el array correctamente ordenado.

Ordenación por Selección

El método de selección consiste en seleccionar el menor elemento del array e intercambiarlo con el elemento ubicado en la primera posición. Entre los elementos restantes, nuevamente buscamos el menor y lo intercambiamos con el elemento que se encuentra en la segunda ubicación, y así sucesivamente hasta tener todos los elementos ordenados.
Una vez leídos los elementos y entrando al algoritmo de ordenamiento en sí, la primera vez que se ejecuta el bucle i hacemos la asignación k=i. Aquí estamos suponiendo que el menor elemento es el primero y corresponde al subíndice 0 (esto es parte del algoritmo de buscar el menor elemento de un array). Luego se inicia el bucle j, que sirve para encontrar el menor elemento de todos los elementos restantes, tomando como valor inicial de comparación al primer elemento.
Al término del bucle j, k contiene el subíndice del menor elemento, el cual será usado para intercambiar la primera ubicación no ordenada mediante el algoritmo de intercambio de dos valores: tmp = x[k]; x[k] = x[i]; x[i] = tmp Estos pasos se repiten hasta tener todos los elementos ordenados. Finalmente, solo nos resta imprimir el array ordenado.

Ordenación por Inserción

El método de inserción consiste en tomar el segundo elemento e insertarlo junto al primero en la posición que corresponda. Luego, se toma el tercer elemento y se le inserta entre los dos elementos en la posición adecuada. El cuarto elemento se inserta entre los tres elementos en la posición adecuada, y así sucesivamente, hasta tomar el último elemento. Tomemos el primer elemento x[0]; luego, los elementos restantes que queramos insertar serán desde i=1 hasta i=n-1, esto es, manejado por el bucle más externo (bucle i) de los bucles anidados. Posteriormente, asignamos el elemento que deseamos insertar a la variable temporal y buscamos la posición correcta donde se insertará. Esto último se hace en el bucle j.
Así, se recorren todos los elementos ya ordenados desde la posición más alta (j=i-1) hasta las más bajas, y mientras el subíndice sea válido (j>=0) y el valor que deseamos insertar (temporal) sea menor que el elemento x[j]. Esto hace que realmente se ubique en la posición correcta.
En cada iteración del bucle j se desplaza el elemento comparado x[j] hacia la posición x[j+1] para hacerle sitio al elemento a insertar (temporal). Al término del bucle j se habrá encontrado la posición adecuada para insertar temporal y, por lo tanto, se realiza la asignación x[j+1] = temporal. Esta secuencia se vuelve a repetir hasta que se hayan insertado en orden los elementos restantes.
